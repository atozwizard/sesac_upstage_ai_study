4언어레벨에서문제 → 인프라수준에서문제Docker 
대부분문제는언어밖에서 발생한다

Windows / macOS / Linux 환경차이로 동일코드가다르게동작함.

서버와 로컬의 OS 차이
서버(Ubuntu)와로컬(macOS)의 동작방식차이때문에문제재현이 어려움.
venv는“언어의존성”만 맞춰줄뿐,
실제문제는OS·시스템라이브러리·외부 서비스처럼 인프라레벨에서 발생한다.
